# Plan - Futuristic Weekly Planner 🚀

A modern, high-performance weekly planning application for Android, designed with a futuristic "Matrix" aesthetic. This app helps users organize their weekly tasks with a smooth 3D-animated interface and keeps them inspired with daily knowledge from Wikipedia.

## ✨ Features

- **3D Pager Navigation:** Smooth, 3D-rotated transitions between days of the week.
- **Matrix Background:** Dynamic, animated "Matrix digital rain" effect for a sci-fi look.
- **Data Persistence:** Integrated with `SharedPreferences` and `JSON` to ensure your plans are saved even after restarting the app.
- **Wikipedia Integration:** Fetches "Today's Featured Article" from Wikipedia API for each specific date.
- **Task Management:** Easy to add, edit, delete, and mark tasks as completed.
- **Dynamic Themes:** Smooth color transitions using Jetpack Compose animations.

## 🛠 Tech Stack

- **Language:** [Kotlin](https://kotlinlang.org/)
- **UI Framework:** [Jetpack Compose](https://developer.android.com/jetpack/compose)
- **Architecture:** Declarative UI pattern
- **Networking:** HttpURLConnection & Wikipedia REST API
- **Local Storage:** SharedPreferences with JSON serialization
